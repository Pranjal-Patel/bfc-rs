# Brainfuck compiler
An actual brainfuck compiler written in rust

### Installation:

``` shell
git clone https://github.com/Pranjal-Patel/bfc-rs
cargo build -- release
```

### Usage:
Here's a brainfuck code to print hello world with a new line
``` brainfuck
>+++++++[<++++++++++>-]<++.
>>++++++++++[<++++++++++>-]<+.
>>++++++++++[<++++++++++>-]<++++++++.
>>++++++++++[<++++++++++>-]<++++++++.
>>+++++++++++[<++++++++++>-]<+.
>>+++[<++++++++++>-]<++.
>>++++++++[<++++++++++>-]<+++++++.
>>+++++++++++[<++++++++++>-]<+.
>>+++++++++++[<++++++++++>-]<++++.
>>++++++++++[<++++++++++>-]<++++++++.
>>++++++++++[<++++++++++>-]<.>

>++++++++++.
```
